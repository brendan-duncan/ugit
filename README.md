# U-Git

Experiment writing a git client as an exercize of vibe coding. If it's wrong or buggy, blame Claude. Developed as a NodeJS Electron application.

## Building

* Install [Node.js](https://nodejs.org) if you don't already have it installed.

* Install node packages by running `npm install` from the project root folder.

* Build the app with `npm run build`.

* Run the app with `npm start`.
